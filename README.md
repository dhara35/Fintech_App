# Fintech App

This is a React Native mobile application for managing fintech-related tasks and services.

## Installation

1. Clone this repository.
2. Navigate to the project directory.
3. Install dependencies using npm or yarn:
   ```bash
   npm install
   # or
   yarn install
   ```

## Usage

- To start the development server:
  ```bash
  npm start
  # or
  yarn start
  ```

- To run the app on Android emulator/device:
  ```bash
  npm run android
  # or
  yarn android
  ```

- To run the app on iOS simulator/device:
  ```bash
  npm run ios
  # or
  yarn ios
  ```

- To run the app in a web browser:
  ```bash
  npm run web
  # or
  yarn web
  ```

## Testing

This project uses Jest for testing. To run tests:
```bash
npm test
# or
yarn test
```
